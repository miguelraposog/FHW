# Reto 02 — Puesta a Punto Low-Cost y Competitiva (UT3 · RA1)

**Contexto:** Has comprado una flota de PCs por **20 €** la unidad. Deben quedar **usables** para un **centro de mayores** (navegación web, correo, videollamada y ofimática online) **sin elevar el precio final**. **En este reto SOLO trabajas la parte de HARDWARE** (diagnóstico, búsqueda de piezas, compatibilidad y presupuesto).

**Escenarios de gasto (elige 1 para tu propuesta final):**
- **S0 (0 €)**: *Sin gasto en hardware.* Solo limpieza, recolocación, y pruebas básicas.
- **S1 (≤ 15 €)**: *Micro-upgrade.* Ej.: SSD 120 GB **de segunda mano** / 4 GB RAM extra reciclada / pasta térmica.
- **S2 (≤ 30 €)**: *Upgrade ligero.* Ej.: SSD 120–240 GB barato, adaptador 2.5"/bahía, pasta térmica, pequeña ampliación RAM.

**Competitividad de mercado (regla):**
`Coste total por unidad = 20 € (compra) + gasto HW + (tarifa interna €/h · horas) ≤ Precio medio de comparables − margen (10–20 %)`.
