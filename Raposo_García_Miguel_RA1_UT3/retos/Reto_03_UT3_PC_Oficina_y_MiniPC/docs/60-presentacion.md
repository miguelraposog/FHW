# Presentación (obligatoria)

Crea una presentación (PowerPoint/Google Slides/LibreOffice Impress).

## Estructura mínima recomendada (8 diapositivas)
1. Portada (título, nombre, grupo, fecha)
2. Objetivo y requisitos (oficina + low cost + compatibilidad)
3. CPU + por qué (compatibilidad)
4. Placa + RAM (compatibilidad)
5. SSD + PSU + chasis (resumen)
6. Presupuesto final (tabla)
7. Mini PC alternativa (pros/contras + precio)
8. Comparación final A vs B (conclusión)

**Regla:** cada diapositiva de producto debe incluir:
- modelo exacto
- precio
- URL (como texto o QR si quieres)
